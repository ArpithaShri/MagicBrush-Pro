# MagicBrush Pro GPU Backend

This is the high-performance GPU backend for MagicBrush Pro, running on FastAPI with Stable Diffusion Inpainting and LPIPS/CLIP scoring.

### Deployment Info
- **SDK**: Docker
- **Hardware**: GPU (T4 or A10G recommended)
- **API Port**: 8000
